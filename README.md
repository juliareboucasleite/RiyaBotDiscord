### RiyaBotDiscord
RiyaBotDiscord é um sistema que combina um bot de Discord com um dashboard web, desenvolvido em Kotlin utilizando Ktor. O projeto permite gerir configurações, comandos e funcionalidades do bot em tempo real através de um painel administrativo, com suporte a configuração por servidor e integração direta entre o backend e o bot. Foi desenvolvido com o objetivo de aprofundar competências em Kotlin, desenvolvimento backend e construção de sistemas integrados.
